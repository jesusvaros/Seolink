## Preguntas Frecuentes

### ¿Cómo se utiliza el microondas Cecotec para fundir chocolate?

Para fundir chocolate en el microondas Cecotec, coloca el chocolate en un bol apto para microondas, selecciona la potencia máxima de 700 W y calienta durante 2 minutos y medio. Si solo necesitas fundir media tableta, reduce el tiempo a 2 minutos. Asegúrate de remover bien para obtener una textura homogénea.

### ¿Qué tamaños incluye el set de cuencos de acero inoxidable Chuner?

El set de cuencos Chuner incluye cuatro tamaños: 0.7 L, 1.2 L, 1.8 L y 2.5 L. Esto te permite medir con precisión los ingredientes para tus recetas, lo que es crucial en la repostería.
